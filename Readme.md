# Stylelint

- Standard config base;
- Prettier;

## Install

```
npm i -D eslint config-ar3secchim
```

<br>

## Usage
Create a `.eslintrc.json` file extending the config:

```
{
  "extends": "config-ar3secchim"
}
```

> You can also use a `.eslintrc.js` instead of JSON if you prefer.
